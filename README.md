# Лаборатория Оракулов

Сайт еженедельных гороскопов, основанных на научных открытиях. Где астрономия встречается с юмором, а физики где-то шутят.

## Структура сайта

- **Главная страница** (`/`) — для пользователей, только чтение гороскопов
- **Админка** (`/admin.html`) — для администратора, загрузка и управление гороскопами

## Для администратора

### Как обновить гороскоп

1. Перейдите в админку: `https://ваш-сайт.ru/admin.html`
2. Введите пароль: `oracle2024`
3. Перетащите CSV файл в область загрузки или нажмите для выбора
4. После успешной загрузки нажмите **"Скачать horoscopes.json"**
5. Замените файл `public/data/horoscopes.json` в проекте
6. Пересоберите и задеплойте сайт:
   ```bash
   npm run build
   ```

### Формат CSV файла

```csv
Знак,Гороскоп
Овен,"Наука недели: Microsoft создает систему хранения данных на стекле.

Гороскоп: На следующей неделе тебе предстоит стать носителем вечной информации..."
Телец,"Наука недели: Устрицы строят рифы с оптимальной геометрией.

Гороскоп: Телец, вселенная намекает: хватит плыть по течению..."
```

**Поддерживаемые названия знаков:** Овен, Телец, Близнецы, Рак, Лев, Дева, Весы, Скорпион, Стрелец, Козерог, Водолей, Рыбы (или на английском).

**Разделители:** запятая (`,`) или точка с запятой (`;`).

**Форматирование:** Текст между `Наука недели:` и `Гороскоп:` выделяется в отдельный блок на сайте.

### Шаблон CSV

Шаблон доступен в файле `public/horoscope-template.csv` или в админке по кнопке "Скачать шаблон".

## Для разработчиков

### Установка

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Сайт будет доступен по `http://localhost:5173`
Админка по `http://localhost:5173/admin.html`

### Сборка

```bash
npm run build
```

Собранные файлы будут в папке `dist/`

### Структура проекта

```
app/
├── public/
│   ├── data/
│   │   └── horoscopes.json    # Данные гороскопов (читаются пользователями)
│   └── horoscope-template.csv  # Шаблон для админа
├── src/
│   ├── hooks/
│   │   ├── usePublicHoroscope.ts  # Hook для чтения (пользователи)
│   │   └── useHoroscope.ts        # Hook для управления (админ)
│   ├── sections/              # Компоненты страниц
│   ├── types/                 # TypeScript типы
│   ├── utils/                 # Утилиты (парсер CSV)
│   ├── App.tsx               # Главная страница (пользователи)
│   ├── AdminApp.tsx          # Админка
│   ├── admin.tsx             # Entry point для админки
│   └── main.tsx              # Entry point для главной
├── admin.html                # HTML для админки
├── index.html                # HTML для главной
└── vite.config.ts            # Конфиг с multiple entry points
```

## Технологии

- React 19 + TypeScript
- Vite 7 (с multiple entry points)
- Tailwind CSS 3
- GSAP (анимации)
- shadcn/ui

## Безопасность

**Важно:** В текущей реализации пароль админа (`oracle2024`) хранится в коде. Для production рекомендуется:

1. Использовать переменные окружения для пароля
2. Добавить серверную авторизацию (JWT)
3. Или использовать внешний сервис авторизации (Auth0, Firebase Auth)
